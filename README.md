
# Expense Tracker UI/UX Design

## Overview
A user-friendly expense tracking application designed to help users manage expenses easily.

## Tools Used
- Figma
- Canva

## Design Process

### User Research
Identified user problems related to expense tracking.

### Wireframes
Created low-fidelity layouts to plan screen structure.

### UI Design
Designed high-fidelity screens with consistent colors and components.

### Prototype
Created interactive user flows in Figma.
